# GeoGebrEmbed plugin for DokuWiki

Embed GeoGebra applets into DokuWiki pages.

![Screenshot](https://user-images.githubusercontent.com/52650214/136657931-be18ce11-58f3-4582-95d2-f4538c23a2c4.png)

All documentation for this plugin can be found at
https://www.dokuwiki.org/plugin:geogebrembed

If you install this plugin manually, make sure it is installed in
lib/plugins/geogebrembed/ - if the folder is called different it
will not work!

Please refer to http://www.dokuwiki.org/extensions for additional info
on how to install extensions in DokuWiki.
